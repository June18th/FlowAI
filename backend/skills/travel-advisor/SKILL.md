# 出行建议生成器

## 功能
根据天气查询结果生成专业的出行建议。

## 输入
- 天气数据（实时 + 未来几日预报）
- 出行目的地城市

## 执行规则

### 1. 天气概览
用简洁的语言总结当天和未来几天的天气状况：温度范围、天气变化趋势、风力等。

### 2. 穿衣建议
根据温度和天气状况推荐合适的着装：
- 高温天气（>30°C）：轻薄透气衣物，防晒装备
- 温暖天气（20-30°C）：短袖 + 薄外套备用
- 凉爽天气（10-20°C）：长袖 + 外套
- 寒冷天气（0-10°C）：毛衣 + 厚外套
- 严寒天气（<0°C）：羽绒服 + 保暖内衣 + 帽子手套
- 雨天：防水鞋具 + 雨伞必备
- 大风天：防风外套 + 帽子固定

### 3. 活动建议
根据天气推荐合适的活动：
- 晴天：户外活动（公园、徒步、骑行等）
- 雨天：室内活动（博物馆、商场、咖啡厅等）
- 多云/阴天：城市观光、美食探索
- 极端天气：建议待在室内，调整行程

### 4. 注意事项
列出需要注意的要点：
- 防晒提醒（紫外线强时）
- 补水提醒（高温时）
- 保暖提醒（低温时）
- 雨具提醒（降雨概率高时）
- 特殊天气预警

## 输出格式
用中文以友好的语气输出，分段落呈现，每部分有小标题。适合展示给终端用户阅读。
